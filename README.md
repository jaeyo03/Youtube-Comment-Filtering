# 유튜브 악플 탐지 웹 사이트

## 🧑🏻‍💻소개
- 유튜브 영상에 존재하는 악성 댓글을 알려주는 웹 사이트 입니다.
- 사용자는 영상의 댓글 중 어떤 댓글이 악성 댓글인지 알 수 있습니다.
  </br>

## 🗂파이프라인 , 🔧사용 기술
- sms-spam-detection 라이브러리를 사용하여 유튜브 댓글 중 악플을 찾고, 악플 데이터만 구글 스토리지에 저장합니다.

- 유튜브 영상을 보여줍니다.
