# 유튜브 악플 탐지 웹 사이트

## 🧑🏻‍💻소개
- 유튜브 영상에 존재하는 악플을 알려주는 웹 사이트 입니다.
- 사용자는 영상의 댓글 중 어떤 댓글이 악플인지 알 수 있습니다.
